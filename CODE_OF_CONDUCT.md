# 社区行为准则

我们希望 Family Tree 社区保持友善、专业、安全，并尊重每个人的隐私和背景。

## 我们鼓励的行为

- 以事实、复现和技术权衡讨论问题；
- 尊重不同经验水平、语言、文化、身份和家庭结构；
- 对反馈保持开放，承认错误并主动修正；
- 在分享日志、截图和样例数据前完成隐私脱敏；
- 将讨论聚焦在项目和可执行改进上。

## 不可接受的行为

- 骚扰、歧视、侮辱、威胁、跟踪或持续的人身攻击；
- 未经允许公开他人的身份、联系方式、家庭关系或其他私人信息；
- 发布露骨内容、垃圾信息，或故意破坏协作环境；
- 利用安全问题伤害用户，或在协调修复前公开可操作的漏洞细节；
- 在项目空间内持续进行与项目无关的煽动性争论。

## 执行

本准则适用于仓库的 Issue、Pull Request、讨论区以及代表项目参与的其他公共空间。维护者可以编辑或移除违规内容，给予警告，临时或永久限制参与，并在必要时向平台报告。

请通过仓库所有者 GitHub 个人资料中公开的联系方式私下报告行为问题；不要在公开 Issue 中披露敏感细节。报告会在合理范围内保密处理。维护者涉及冲突时，应由未涉及的维护者处理。
